Task:

Given the array of five integers.

Find maximum and minimum possible sum that can be calculated by summing precisely 4 of 5 elements of the array.

Print the corresponding values to stdout in "minimum: x, maximum: y."

Write unit tests to check if your code works correctly with different arrays (at least five data sets). For tests, use library testify (https://github.com/stretchr/testify)
